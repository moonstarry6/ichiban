需要先在 Google Apps Script 建立 Web App,將網址貼到這裡。

Apps Script 設定教學
1. 開啟 Google 試算表,選單「擴充功能 → Apps Script」
2. 將此範本程式碼貼到編輯器並儲存
3. 點「部署 → 新增部署作業 → 類型:Web 應用程式」
4. 執行身分選「我」,存取權選「任何人」
5. 複製部署後的網址貼到上方欄位並儲存
